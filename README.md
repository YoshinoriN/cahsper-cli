# [WIP]: cahsper-cli

CLI tool for [Cahsper](https://github.com/YoshinoriN/cahsper).

# Requirements

* go 1.15

# Commands

|command||
|---|---|
|`help`, `-h`, `--help`|Help about any command.|
|`version`|Show version number.|
|`config list`|Show cahsper configure variables.|

# Build

```sh
$ go build
```

# Test

```sh
$ go test ./...
```
